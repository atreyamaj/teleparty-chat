# Teleparty Chat App

A React + TypeScript SPA built for the Teleparty Full-Stack Engineer Challenge.

## Tech Stack

*   React 19 + TypeScript
*   Vite
*   MUI v5
*   Zustand + Immer
*   @tanstack/react-virtual
*   React Hook Form + Zod
*   `teleparty-websocket-lib`
*   Vitest + @testing-library/react + MSW
*   ESLint + Prettier
